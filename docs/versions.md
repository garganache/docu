## Adding New Versions

When a new version is released:

1. Documentation is added to `docs/{app}/{version}/`
2. Sidebar is updated with new version
3. This index page is updated
4. Old versions remain accessible for reference

## Support Policy

- **Stable versions**: Full support for 12 months
- **Legacy versions**: Security fixes only
- **EOL versions**: No support, docs archived
