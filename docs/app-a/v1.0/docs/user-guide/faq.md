# Frequently Asked Questions

Quick answers to common questions about **App A**.

## Getting Started

### How do I create an account?

Visit the [signup page](https://app-a.example.com/signup) and enter your email, password, and organization name. You'll receive a confirmation email within minutes.

### Is there a free trial?

Yes! All new accounts get a **14-day free trial** with full access to all features. No credit card required.

### Can I import data from other tools?

Yes, we support imports from:
- CSV files
- Trello
- Asana
- Jira

See our [import guide](#) for details.

## Pricing & Billing

### What plans are available?

- **Free** - Up to 5 users, basic features
- **Pro** - $10/user/month, all features
- **Enterprise** - Custom pricing, SSO, SLA

### Can I change my plan later?

Yes, you can upgrade or downgrade anytime. Changes take effect immediately, and we'll prorate your billing.

### Do you offer discounts?

- **Annual billing** - Save 20%
- **Nonprofits** - 50% off
- **Education** - Free for students and teachers

## Usage

### How many projects can I create?

- **Free plan** - 3 projects
- **Pro plan** - Unlimited
- **Enterprise** - Unlimited

### What file types are supported?

We support all common file types:
- Documents: PDF, DOC, DOCX, TXT
- Images: JPG, PNG, GIF, SVG
- Spreadsheets: XLS, XLSX, CSV
- Others: ZIP, MP4, MP3

### Is there a file size limit?

- **Free plan** - 50 MB per file
- **Pro plan** - 500 MB per file
- **Enterprise** - Custom limits
